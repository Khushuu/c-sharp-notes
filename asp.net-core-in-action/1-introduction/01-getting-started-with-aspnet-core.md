# Getting Started with ASP.NET Core

## Chapter scope
- **What is ASP.NET Core?**
- **Things you can build with ASP.NET Core**
- **How ASP.NET Core works**

## Why this chapter matters
- Choosing to learn and develop with a new framework is a **big investment**.
- It’s important to establish early on whether ASP.NET Core is **right for you**.
- This chapter provides background on ASP.NET Core:
  - **what it is**
  - **how it works**
  - **why you should consider it** for web applications

## Outcome
- By the end of this chapter, you should have:
  - a good overview of the **benefits of ASP.NET Core**
  - an understanding of the **role of .NET 7**
  - familiarity with the **basic mechanics** of how ASP.NET Core works
